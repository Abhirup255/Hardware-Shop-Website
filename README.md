# 🛒 Hardware Shop Website

This project is a responsive Hardware Shop Website developed using HTML, CSS, and JavaScript. It simulates an online platform where users can browse hardware products, view details, and interact with the interface.

## 📌 Features
- Responsive and user-friendly design
- Product listing with details
- Easy navigation using menu bar
- Interactive elements using JavaScript
- Clean and structured UI layout

## 🛠️ Technologies Used
- HTML5 – Structure of web pages
- CSS3 – Styling and layout design
- JavaScript – Interactivity and dynamic behavior

## 📂 Project Structure
Hardware-Shop-Website/
- index.html
- css/style.css
- js/script.js
- images/

## 🚀 How to Run
1. Download or clone the repository
2. Open the index.html file in any web browser
3. Explore the website features

## 🎯 Objective
The objective of this project is to apply front-end development skills to build a real-world web interface and improve understanding of UI design and user interaction.
